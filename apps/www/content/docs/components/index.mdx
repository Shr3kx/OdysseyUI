# Interfaces that breathe

Odyssey UI is a curated kit of animated, composable components built for shipping. Every piece drops into your codebase ready to work, then bends to your design without fighting the API or untangling style overrides.

Landing page, dashboard, or full app shell—build expressive interactions with tools you already know: React, TypeScript, Tailwind CSS.

## What's here?

- Carefully grouped component families with practical examples.
- Installation paths that work for both quick copy-paste and deeper integration.
- Motion-forward UI patterns that still prioritize readability and structure.

Start with the collection that matches your current priority, then expand from there as your interface grows.
