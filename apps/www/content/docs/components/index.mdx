# Build interfaces that feel alive

Odyssey UI is a curated collection of animated, composable components built for real products. Every piece is designed to be dropped into an existing codebase quickly, then customized without fighting the API or style layer.

If you are building a landing page, a product dashboard, or a fully documented app shell, these components help you ship expressive interactions while keeping your stack familiar: React, TypeScript, and Tailwind CSS.

## What you get on this page

- Carefully grouped component families with practical examples.
- Installation paths that work for both quick copy-paste and deeper integration.
- Motion-forward UI patterns that still prioritize readability and structure.

Start with the collection that matches your current priority, then expand from there as your interface grows.

## Recommended path

Begin with **Buttons** for actions and feedback, continue with **Community** patterns for social or collaborative UI, then explore broader Odyssey examples for more advanced animated composition. This order gives you fast visual wins first, followed by deeper layout and interaction patterns.
